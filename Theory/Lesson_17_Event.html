Lesson Theory

Event

Событие (event) — это сигнал от браузера о том, что что-то произошло:

  • пользователь кликнул мышью,
  • навёл курсор,
  • нажал клавишу,
  • отправил форму,
  • загрузилась страница,
  • изменилось содержимое input и т.д.

JavaScript предоставляет возможность "слушать события" и выполнить заранее написанный код, когда событие произошло. Для этого мы пишем функцию - обработчики событий (event handlers).

Например:

<button id="btn">Нажми меня</button>

<script>
  const btn = document.getElementById("btn");

  btn.addEventListener("click", () => {
    alert("Кнопка нажата!");
  });
</script>

Здесь:

  • событие → click
  • элемент → кнопка
  • обработчик → стрелочная функция

События бывают разные:

* мышь (`click`, `mousemove`, `mouseenter`…)
* клавиатура (`keydown`, `keyup`)
* формы (`submit`, `input`, `change`)
* окно и документ (`load`, `scroll`, `resize`, `beforeunload`)

Событие — это объект (event) Когда происходит событие, браузер создаёт объект event, в котором хранится вся информация:

* какая клавиша нажата,
* какие координаты мыши,
* на каком элементе произошло,
* и многое другое.

btn.addEventListener("click", (event) => {
  console.log(event.type); // "click"
  console.log(event.target); // <button>
});

Всплытие событий Если нажать на кнопку внутри div, то событие «поднимается вверх» к родителям (это называется bubbling). Благодаря этому можно вешать обработчики не только на сам элемент, но и на его предка. Например: в форме несколько input, каждое из них может изменить значение (change). Но мы можем повесить обработчик только на родительский элемент (form). При этом информацию о том, какое именно поле породило событие, можем получить из объекта event.target

Как вешать обработчики

// Способ 1: атрибут в HTML
<button onclick="alert('Привет')">Click me</button>

// Способ 2: свойство DOM-объекта
btn.onclick = () => alert("Привет!");

// Способ 3: addEventListener (современный способ)
btn.addEventListener("click", () => alert("Привет!"));

Категория Событие Когда возникает Пример

mouseleave Ушли курсором (не всплывает) el.addEventListener("mouseleave", () => console.log("Вышли с элемента"));
mouseover Навели курсор (всплывает) el.addEventListener("mouseover", () => console.log("Навели на элемент/дочерний"));
mouseout Увели курсор (всплывает) el.addEventListener("mouseout", () => console.log("Увели курсор"));

keyup Клавиша отпущена document.addEventListener("keyup", e => console.log("Отпустили", e.code));
keypress (устаревший) Символьная клавиша нажата ❌ лучше использовать keydown
Формы submit Отправка формы form.addEventListener("submit", e => {e.preventDefault(); console.log("Форма отправлена");});
input Изменение поля ввода в реальном времени input.addEventListener("input", () => console.log(input.value));
change Изменение значения (после потери фокуса или выбора) select.addEventListener("change", () => console.log("Выбрано значение"));
focus Элемент получил фокус input.addEventListener("focus", () => console.log("Фокус на input"));
blur Потеря фокуса input.addEventListener("blur", () => console.log("Потеря фокуса"));
Окно / Документ load Страница полностью загружена window.addEventListener("load", () => console.log("Страница загружена"));
DOMContentLoaded HTML загружен (без картинок и стилей) document.addEventListener("DOMContentLoaded", () => console.log("DOM готов"));
resize Изменение размера окна window.addEventListener("resize", () => console.log(window.innerWidth));
scroll Прокрутка window.addEventListener("scroll", () => console.log(window.scrollY));
beforeunload Попытка покинуть страницу window.addEventListener("beforeunload", e => {e.preventDefault(); e.returnValue = "Уверены?";});
Drag & Drop dragstart Начало перетаскивания el.addEventListener("dragstart", () => console.log("Перетаскивание начато"));
dragover Элемент "над" зоной el.addEventListener("dragover", e => e.preventDefault());
drop Элемент брошен zone.addEventListener("drop", () => console.log("Элемент отпущен"));

Поведение по умолчанию

Браузер для многих событий имеет встроенное поведение. Например:

  • при клике по ссылке <a> — переход по href,
  • при отправке формы <form> — перезагрузка страницы,
  • при правом клике — открытие контекстного меню,
  • при dragover — запрет на сброс элементов.

Чтобы это поведение не происходило, вызывают preventDefault(). event.preventDefault() — это метод объекта события в JavaScript, который отменяет действие браузера по умолчанию.

element.addEventListener("событие", function(event) {
  event.preventDefault();
});

Примеры

Отменяем переход по ссылке

<script>
  document.getElementById("link").addEventListener("click", function(e) {
    e.preventDefault(); //  блокируем переход
  });
</script>

Блокируем отправку формы

<script>
  document.getElementById("myForm").addEventListener("submit", function(e) {
    e.preventDefault(); // не перезагружаем страницу
    ....
  });
</script>
