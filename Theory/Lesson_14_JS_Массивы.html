JS Intro

Массивы в JavaScript

Массив в JavaScript — это объект, он может содержать элементы любого типа, включая другие массивы и объекты.

let fruits = ['apple', 'banana', 'cherry'];

Особенности массивов в JS

Особенность Описание
Гибкая типизация В одном массиве могут быть значения разных типов
Автоматическое изменение длины Массив может автоматически увеличиваться
Динамические индексы Можно добавлять значения по любым индексам, включая "дырки"
Основан на объектах Массив — это объект с числовыми ключами и специальными свойствами
Специальные методы Имеет богатый набор встроенных методов для обработки

Объявление массива

Массивы в JavaScript можно объявить двумя способами:

let fruits = ['apple', 'banana', 'cherry']; // предпочтительный способ
let numbers = new Array(1, 2, 3);           // тоже работает, но используется реже

Индексация и длина массива

let arr = ['a', 'b', 'c'];
console.log(arr[0]);      // 'a'
console.log(arr.length);  // 3

Индексация начинается с 0.

Добавление/Удаление элементов

let arr = [1, 2];
arr.push(3);    // [1, 2, 3]
arr.pop();      // [1, 2]
arr.unshift(0); // [0, 1, 2]
arr.shift();    // [1, 2]
arr[3] = 10;    // [1,2,10]

Функции

Создание функций

  1. Function Declaration — как привычное объявление метода.
function greet(name) {
    return `Привет, ${name}`;
}

Вызвать функцию, объявленную таким образом, можно вызвать до объявления.

Параметры и возвращаемое значение

В отличие от JAVA в JS у функции параметры и результат не типизированные

Например:

function add(a, b) { return a + b; }

В данную функцию можно передать любые параметры

add(10,20) // 30
add('hello',' js') // 'hello js' 
add(10) // NaN

и может вернуть что угодно, и даже если передали строки — он попытается их сложить, и вернуть результат.
Если не передать параметр, его значение будет undefined

Параметры по умолчанию

function greet(name = "гость") {
  return `Привет, ${name}!`;
}

console.log(greet()); // Привет, гость!
  • Значение параметра будет использовано,
  • если аргумент undefined или не передан

Функция, как данные

В JS функция - это объект.
В JS можно:

  • Передать функцию в аргумент
  • Вернуть функцию из функции
  • Хранить в массиве/объекте

Например: 

function greetUser(name, func) {
    console.log(`Привет, ${name}!`);
    func();
}

function sayHowAreYou() {
    console.log("Как дела?");
}

greetUser("Анна", sayHowAreYou);

// Вывод:
// Привет, Анна!
// Как дела?

в этом примере функция sayHowAreYou является параметром для функции greetUser. Мы можем в greetUser передать любую функцию, тем самым "на лету" изменять поведение функции greetUser. Говорят, sayHowAreYou - колбэк функция. Колбэк — это функция, которую мы передаём в другую функцию как аргумент, чтобы та вызвала её при необходимости.
